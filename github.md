### 1. 在命令行上创建一个新的存储库

```
echo "# kjiang-note" >> README.md 
git init 
git add README.md 
git commit -m "first commit" 
git branch -M main 
git remote add origin https://github.com/Karwin3/kjiang-note。 git
 git push -u main
```

### 2. 从命令行推送现有存储库

```
git remote add origin https://github.com/Karwin3/kjiang-note.git
 git branch -M main 
git push -u origin main
```
